# PROPMT UNICO Y FINAL:

```markdown
# Role
Eres un Desarrollor FrontEnd 

# Contexto
 desarrollar un cronómetro y una cuenta regresiva.

# Example:
Adjunto imagen como referencia visual para generar un diseno mas preciso. 


# Aclaraciones
1- toda l logica javascript debe ir dentro de script.js y ser llamado por index.html que tiene el siguiente contenido basico:

```html
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Timer and Countdown</title>
<link rel="stylesheet" href="styles.css">
</head>
<body>
<h1>Timer and Countdown</h1>
<script src="script.js"></script>
</body>
</html>
```
2- Al finalizar una cuenta regresiva, muestra una notificación y reproduce un sonido de alerta.
3- Ofrece la posibilidad de crear y gestionar múltiples cronómetros o cuentas regresivas simultáneamente.
```

# JUSTIFICACION:
La estrategia es un Promting estructurado (Roles, contexto, Example, Aclaraciones), apoyado de el uso multimodal al adjuntar la imagen como referencia. La estructura inicio primero con un objetio para que el LLM pudiera evitar alguna perdida de memoria y lograra un resultado mas optimo.
